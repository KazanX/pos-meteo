Detailed Description

Weather station system.

App running on Arduino UNO. It's gathering data from sensors and sending them to database located on remote server.

List of sensors:

  1. Thermometer
  2. Barometer
  3. Humidity sensor
  4. Anemometer
  5. Wind vane
